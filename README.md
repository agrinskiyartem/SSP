# Учебный проект «Банкоматы» (PHP + MySQL)

## Быстрый запуск (XAMPP/OpenServer)
1. Скопируйте папку проекта в `htdocs` (XAMPP) или в каталог домена (OpenServer).
2. Импортируйте файл `schema.sql` в MySQL (phpMyAdmin или консоль):
   ```sql
   SOURCE /path/to/schema.sql;
   ```
3. Проверьте настройки в `config/db.php` (хост, пользователь, пароль).
4. Откройте в браузере `http://localhost/SSP`.

## Тестовые аккаунты
- Администратор: `admin / admin123`
- Оператор/аналитик: `operator / operator123`

## Основные возможности
- Авторизация и роли (администратор и оператор/аналитик).
- CRUD справочников (банки, банкоматы, клиенты, карты).
- Снятие наличных с расчётом комиссии.
- Журнал операций с фильтрами.
- Аналитические отчёты.
- Демонстрация транзакций и блокировок `SELECT ... FOR UPDATE`.
